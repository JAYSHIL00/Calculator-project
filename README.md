# Simple Calculator

## Description
A basic Windows Forms desktop application built using C# and the .NET Framework. It serves as a simple calculator to perform standard arithmetic operations.

## Features
- Provides functionality to perform basic arithmetic operations: Addition (+), Subtraction (-), Multiplication (*), and Division (/).
- Handles invalid input format such as letters instead of numbers.
- Handles division by zero gracefully using error messages.
- Has a 'Clear' button to easily reset inputs and the result label.
- Simple, clear graphical user interface.

## Tools Used
- C# Programming Language
- .NET Framework (Windows Forms)
- Visual Studio

## Steps to Run the Project
1. Open **Visual Studio**.
2. Click on **Open a project or solution**.
3. Navigate to the folder containing this project and double-click `Calculator.csproj` (or import the respective C# files if you are creating a project from scratch).
4. Once the solution is loaded, ensure the build target is set properly (e.g., Debug/Any CPU).
5. Press **Start** (or hit `F5` on your keyboard) to compile and run the application.
6. The *Simple Calculator* window will appear. Enter your numbers in the two text fields and click any of the operation buttons to see the result!
